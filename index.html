Privacy Policy — Color Rain

Last Updated: 26.11.2025

Color Rain (“the Game”), developed by Batuhan KANBER, respects your privacy. This Privacy Policy explains how the Game collects, uses, and protects your information.

1. Information We Collect

The Game itself does not collect or store any personal data from users. However, the Game uses certain local features and third-party services described below.

1.1 Local Storage (High Score Data)

The Game uses local storage to save gameplay-related data such as high scores and basic game progress. This data is:

  • Stored only on your device
  • Not uploaded, shared, or accessed by Batuhan KANBER
  • Removed only if the user clears app data or uninstalls the game

1.2 AdMob (Google AdMob)

The Game uses Google AdMob to show advertisements. AdMob may automatically collect information such as:

  • Device information (model, OS, identifiers)
  • IP address
  • Approximate location (for ads)
  • App usage data
  • Ad interaction data

Google’s Privacy Policy applies to all data processed by AdMob: https://policies.google.com/privacy

2. How the Information Is Used

Any data collected by third-party services (such as AdMob) may be used for:

  • Showing personalized or non-personalized ads
  • Limiting repeated ads
  • Improving ad quality
  • Measuring ad performance
  • Fraud prevention and security

Batuhan KANBER does not access or store this data. Local storage data is used only to keep your high score.

3. Third-Party Services

The Game may use the following service: Google AdMob (advertising). Each provider has its own privacy policy.

4. Children’s Privacy

The Game is not intended for children under 13. If you believe your child interacted with ads that collected information, please contact us.

5. Data Security

The Game itself does not process personal data. Local storage is fully contained on the device. Any data collected by AdMob is managed by Google’s secure systems.

6. Changes to This Policy

We may update this Privacy Policy periodically. Changes will be posted on this page with the updated date.
